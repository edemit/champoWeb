La pomme

Généralités

Généralement, on distingue trois types de pommes alimentaires :
  • les pommes à cidre
  • les pommes de table
  • les pommes à cuire

Wikipedia

Etymologie

Le mot pomme est issu du gallo-roman POMA, terme d'origine latine avec la désinence -a (pluriel neutre collectif), prise pour un féminin singulier
